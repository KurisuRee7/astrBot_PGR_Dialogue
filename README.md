# AstrBot 战双text插件

## 简介

当用户发送的消息中包含战双的部分角色名称时，机器人有概率会自动回复一段相关文本。

## 功能

*   **关键词触发：** 检测消息中是否包含战双的部分角色名称。
*   **自动回复：** 当检测到关键词时，自动回复指定文本。

## 使用方法

检测到战双的部分角色名称时，机器人就会自动回复。

## 配置

本插件无需任何配置。

## 注意事项

*   本插件需要 AstrBot 机器人环境。
*   请确保 AstrBot 机器人已正确安装并配置。

## 示例

**用户：** 该打战双了

**机器人：** 这就是首席的实力
